🛏️ 智能婴儿床控制系统 · 第一代原型

🧸 项目背景

在照顾婴儿的过程中，父母有时因做饭或短时离开而无法实时照看婴儿。尤其是独自带娃的情况，如有突发状况（如婴儿翻出床外），可能会造成严重后果。为了解决这一安全隐患，本项目设计并实现了一款以“安全看护”为目标的智能婴儿床系统。

目前市面上的婴儿床多以提升舒适度、助眠为主（如电动摇床、语音安抚），而缺乏对婴儿状态的感知与远程提醒功能。我们的系统正是以此为突破口。

🎯 项目目标

  • 实现婴儿状态的实时监测与报警提醒；
  • 集成多种传感器实现环境信息采集；
  • 使用 LCD 显示屏进行信息输出；
  • 提供初代智能控制系统软硬件实现框架。

💡 系统组成

🖥️ 软件环境

  • Windows 10
  • Arduino IDE

🔩 硬件组件

硬件名称 数量 说明
Arduino UNO 1 控制主板
面包板 2 原型电路搭建
杜邦线 多根 电路连接
温湿度传感器 1 DHT 系列传感器
声音传感器 1 检测婴儿哭声
超声波传感器 1 距离检测，防止翻身摔落
LCD 显示屏（1602a） 1 实时数据显示（温湿度等）

⚙️ 功能实现

🔎 信息感知模块

  • 声音传感器：检测婴儿是否哭泣；
  • 温湿度传感器：监测环境温度与湿度；
  • 超声波模块：检测婴儿位置、姿态变化，防跌落。

📺 LCD 显示模块（1602a）

实现功能包括：

  • 显示温度、湿度实时数据；
  • 显示报警信息；
  • 支持字符、字符串打印；
  • 转换数字为字符以便于显示。

LCD 显示控制部分的部分核心代码如下：

int LCD1602_RS = 7;
int LCD1602_EN = 6;
int DB[4] = { 2, 3, 4, 5 };

void LCD_Command_Write(int command) { /* 写命令函数 */ }
void LCD_Data_Write(int dat) { /* 写数据函数 */ }
void LCD_SET_XY( int x, int y ) { /* 设置光标位置 */ }
void LCD_Write_Char( int x, int y, int dat) { /* 写单个字符 */ }
void LCD_Write_String(int X, int Y, char  *s) { /* 写字符串 */ }
void zhuan (int n,char *s,char *c) { /* 数字转字符 */ }

👉 使用 LiquidCrystal_I2C lcd(0x27,16,2); 进行 LCD1602a 初始化配置。

🧪 项目运行效果

  • LCD 显示屏可以清晰显示采集到的温度、湿度等环境信息；
  • 系统响应速度良好；
  • 各项传感器运行稳定，适合部署原型系统。

🧠 项目亮点

  • 🌡️ 温湿度感知，便于判断环境是否舒适；
  • 👶 声音感应，及时捕捉婴儿哭声；
  • 📏 距离检测，防止翻出床外；
  • 🔁 基于 LCD 的交互式反馈；
  • 🛠️ 嵌入式系统构建 + 软件开发于一体。

🚀 项目后续升级方向

  • 接入 WiFi 模块实现远程数据推送与控制；
  • 添加摄像头模块，支持视频监控；
  • 利用 APP 实现数据可视化与远程操作；
  • 实现自动安抚机制，如自动摇床、播放安抚音频等。

📚 结语

本项目是一款面向家庭实际场景的嵌入式智能硬件初步尝试，聚焦婴儿安全看护。虽然仍处于第一代原型阶段，但其整体构思清晰、功能合理、硬件设计可扩展，具备良好的发展潜力。
