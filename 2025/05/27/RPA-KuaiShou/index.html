🤖 快手一键批量取消脚本 - RPA 自动化流程机器人

📍 作者： Jia’nan Zhao
📅 时间： 2024‑3‑15

🎯 bilibili项目演示地址：
🤖 快手一键批量取关
🤖 快手一键批量取消点赞

本项目基于 RPA 自动化流程技术，提供一键执行脚本，支持在快手平台上批量取消以下三类用户行为：

  • 👤 已关注账号
  • 👍 点过赞的视频
  • 📌 收藏夹内容

该脚本大幅提升人工操作效率，适用于账号清理、隐私保护、内容管理等场景。

📋 功能概览

功能模块 描述
自动登录 自动输入账号密码，登录快手账号
一键取消关注 快速批量取消当前账号已关注的所有用户
一键取消点赞 自动进入点赞列表，逐条取消点赞
一键取消收藏 遍历收藏内容并自动取消所有收藏
滑动操作模拟 模拟用户操作滚动页面，保证页面加载完成
异常场景处理 针对页面卡顿、接口延迟等异常，设定重试机制

🧭 流程图预览

graph TD
  A[启动脚本] --> B[登录快手账号]
  B --> C{是否需要验证码？}
  C -- 是 --> D[人工完成]
  C -- 否 --> E[进入功能模块选择]
  D --> E
  E --> F[取消关注]
  E --> G[取消点赞]
  E --> H[取消收藏]
  F --> I[模拟滚动加载]
  G --> I
  H --> I
  I --> J[逐条取消]
  J --> K[保存日志]
  K --> L[流程结束]
`

⚙️ 使用指南

1️⃣ 环境准备

  • Python 3.8+
  • 安装依赖库：
  • 建议使用稳定网络环境，避免快手接口阻断

2️⃣ 配置账号

在 config.yaml 中填写快手账号及登录信息：

3️⃣ 启动脚本

根据命令行提示选择操作功能：

  • 输入 1：一键取消所有关注

  • 输入 2：一键取消所有点赞

  • 输入 3：一键取消所有收藏（原理同上取消所有点赞）

📁 项目结构

💡 功能亮点

  • 🧠 智能滑动模拟，确保页面元素加载完全
  • ⏱ 高效循环操作，几分钟内清空数百项数据
  • 📄 全程生成操作日志，方便结果溯源与排查
  • 🔄 支持中断续执行与异常恢复

❗ 注意事项

  • 登录时如遇滑动验证码，请手动辅助完成，脚本会继续运行
  • 若账号存在安全限制（如频繁操作），请适当降低脚本运行频率
  • 本脚本仅供学习研究使用，禁止用于违反平台规则的行为

🚀 未来优化方向

  • ✅ 接入验证码识别 API，实现全程无人值守
  • 📱 兼容移动端浏览器协议，支持 App 模拟
  • 🧩 增加选择性取消（如取消近7天点赞内容）
