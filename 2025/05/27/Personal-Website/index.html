🚀 部署到 Vercel

  1. 登录 Vercel 并使用 GitHub 账号授权。
  2. 点击 “Add New...” -> “Project”，导入之前创建的 GitHub 仓库。
  3. 选择 “Other” 作为框架预设，保持其他设置默认，点击 “Deploy”。
  4. 部署完成后，您将获得一个形如 yourproject.vercel.app 的域名。(Tangly的学习笔记 | 取势、明道、优术, CSDN博客)

🌍 绑定自定义域名（可选）

  1. 在 Vercel 项目的 “Settings” -> “Domains” 中添加您的自定义域名。
  2. 根据提示，在域名注册商的 DNS 设置中添加相应的解析记录（如 CNAME 或 A 记录）。
  3. 等待 DNS 生效后，您的博客即可通过自定义域名访问。(Tangly的学习笔记 | 取势、明道、优术, CSDN博客)
