🌿 通海无人机数据标注

📍 作者： Jia’nan Zhao
📅 时间： 2024‑10‑18

🎯 bilibili项目演示地址： 🌿 通海无人机数据标注

本项目致力于利用无人机获取的高分辨率农田图像，对不同类型的农作物地块进行不规则多边形标注，用于构建智能农业场景下的图像识别与分类模型数据集。

🛰️ 项目背景

无人机遥感影像技术在智慧农业中日益重要，尤其在农作物分类、长势评估与产量预测方面。为了提升模型识别准确度，我们使用 Labelme 工具对下列农作物区域进行了精准多边形标注：

  • 大棚
  • 玉米
  • 芋头
  • 片菜
  • 包菜
  • 甘蓝
  • 屏障
  • 蒲瓜
  • 大棚瓜果

🖼️ 标注样例展示

图片来自无人机拍摄的真实的无人机航拍高清农田区域。

  • ✅ 每一块农田区域使用不同颜色的不规则多边形进行标注
  • ✅ 标签列表位于右侧，可进行复选查看隐藏
  • ✅ 标注工具使用的是 Labelme + EfficientSAM（AI辅助模式）

🧰 标注工具介绍对比

工具名称 LabelImg Labelme
标注形状 仅支持矩形框 ✅ 支持任意多边形
适用场景 物体检测（如YOLO） 语义分割、多边形标注（如Mask R-CNN）
输出格式 XML（Pascal VOC）、TXT（YOLO） JSON（COCO格式可转换）
AI辅助标注 ❌ 不支持 ✅ 支持 EfficientSAM
跨平台 ✅ 支持 ✅ 支持
使用门槛 较低，操作简单 中等，适合专业标注任务
安装方式 pip install labelImg pip install labelme

推荐：

  • LabelImg：适用于简单的目标检测任务
  • Labelme：适用于精细的区域标注、遥感、医学、农田等专业图像任务

🗂️ 数据结构示意

Dataset/
├── input_data/
│   ├── DJI_20240912122720_0595.JPG        # 原始无人机图像
│   └── DJI_20240912122720_0595.json       # Labelme 标注文件
├── labels_summary.xlsx                    # 标签汇总表
└── readme.md                              # 项目说明文档

📌 标注建议与规范

  • 使用“创建多边形”功能，沿着作物边界精细描点
  • AI 模型可选：EfficientSAM (accuracy)，可提高半自动标注效率
  • 避免冗余重叠区域，确保多边形闭合
  • 每块地一类标签，统一标签命名，确保一致性
  • 建议：每类农作物不少于 50 个标注区域

💡 后续用途说明

标注数据可用于以下任务：

  • 🌾 农作物分类模型训练（如 Swin-Unet、DeepLabV3+）
  • 🛰️ 农业遥感影像分割与分析
  • 🤖 多模态智慧农业数据融合项目
