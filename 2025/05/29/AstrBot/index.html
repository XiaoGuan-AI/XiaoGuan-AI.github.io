🌌 AstrBot：多平台 LLM 聊天机器人部署

✨ 跨平台部署 | 多模型支持 | 插件扩展 | 可视化管理 | 一键部署 (OpenI)
📍 原作者： Sorater
📍 二次部署： Jia’nan Zhao
📅 时间： 2025‑5‑29

🎯 bilibili原作者演示地址： 🌌 AstrBot：多平台 LLM 聊天机器人

📌 项目简介

AstrBot 是一个开源的多平台聊天机器人及开发框架，支持多种大语言模型（如 OpenAI GPT、Google Gemini、Llama 等）和多种消息平台（如 QQ、Telegram、微信等）。AstrBot 提供多轮对话、语音转文字、网页搜索等功能，配备代码执行器和可视化管理面板，方便用户配置和扩展。其模块化设计支持插件开发，可一键部署到多种环境（如 Docker、Windows、Replit 等）。(AI工具集)

  • 兼容 QQ（OneBot）、QQ频道、微信（Gewechat、企业微信）、Telegram、飞书等主流消息平台。
  • 这里展示的是在QQ部署的私聊。
  • QQ或微信群聊，有待部署开发。。。（主要是没钱开云服务器部署，哈哈哈）

📊 项目展示

1. AstrBot 仪表盘

该仪表盘展示了 AstrBot 的主要控制界面，用户可以在此配置和管理本地部署的大模型服务。界面中可能包括模型加载状态、服务提供商配置、API 接口信息等，便于用户实时监控和调整模型运行参数。

2. NapCat 仪表盘

NapCat 的仪表盘提供了机器人的管理界面，用户可以在此查看机器人状态、配置聊天参数、查看消息日志等。通过该界面，用户能够方便地与本地部署的模型进行交互，实现个性化的聊天体验。

🧠 人格与模型切换示例说明

小野猫人格 切换大模型供应商示例 切换人格示例 暖男人格

  • 小野猫人格：展示了机器人在“小野猫”人格设定下的聊天界面，可能表现出调皮、亲昵的语气风格，适合营造轻松愉快的对话氛围。

  • 切换大模型供应商示例：该截图可能展示了如何在 AstrBot 中更换底层的大模型服务提供商，例如从 OpenAI 切换到本地部署的 DeepSeek 模型，以满足不同的性能和隐私需求。

  • 切换人格示例：展示了在 NapCat 中切换机器人人格设定的操作界面，用户可以根据需要选择不同的预设人格，如“猫娘”、“御姐”等，改变机器人的对话风格。

  • 暖男人格：展示了机器人在“暖男”人格设定下的聊天界面，可能表现出体贴、关怀的语气风格，适合提供安慰和支持的对话场景。

WebUI 版本：v3.5.11

📦 系统指令

指令 功能描述
/plugin 查看插件列表与帮助信息
/t2i 开关 文本转图片 功能
/tts 开关 文本转语音 功能
/sid 获取当前会话 ID
/op 管理员指令面板（需权限）
/wl 白名单管理
/dashboard_update 更新管理面板（管理员专用）
/alter_cmd 设置指令权限（管理员专用）

🤖 大模型相关指令

指令 功能描述
/llm 开启/关闭大语言模型
/provider 查看/设置大模型提供商
/model 查看可用模型列表
/ls 查看对话列表
/new 创建新对话
/groupnew 群号 为群聊创建新对话（管理员）
/switch 序号 切换到指定对话
/rename 新名字 重命名当前对话
/del 删除当前对话（管理员）
/reset 重置当前 LLM 会话
/history 查看当前会话记录
/persona 设置人格情景（管理员）

🛠️ 工具/拓展指令

指令 功能描述
/tool ls 查看可用函数工具列表
/key 设置或查看 API Key（管理员）
/websearch 执行网页搜索（已美化为 Markdown 格式）

🚀 核心功能

🧠 多语言模型支持

  • 支持接入 OpenAI、DeepSeek、Google Gemini、ChatGLM、Llama、Ollama、LLMTuner 等多种大语言模型。
  • 支持本地部署模型，满足不同场景需求。(AI工具集, GitHub)

💬 多平台接入

  • 兼容 QQ（OneBot）、QQ频道、微信（Gewechat、企业微信）、Telegram、飞书等主流消息平台。
  • 未来将支持钉钉、Discord、WhatsApp、小爱音响等平台。(AI工具集, GitHub)

🧩 插件扩展机制

  • 提供优化的插件系统，支持开发者轻松扩展功能，降低开发难度。
  • 已支持安装多个插件，实现功能多样化。(OpenI, GitHub)

🛠️ Agent 能力

  • 原生支持代码执行器、自然语言待办事项、网页搜索等功能。
  • 可与 Dify 平台对接，实现智能助手、知识库和工作流的便捷接入。(AI工具集, OpenI)

🎛️ 可视化管理面板

  • 配备可视化管理面板，支持配置修改、插件管理、日志查看等功能。
  • 集成 WebChat 功能，用户可以直接与大模型进行对话。(AI工具集, OpenI)

🎙️ 多模态交互体验

  • 支持图片理解和语音转文字（Whisper），具备多轮对话和人格情境功能。
  • 支持 TTS（文本转语音）和 STT（语音转文本），实现语音输入与输出。(AI工具集)

⚙️ 部署与配置

🖥️ Windows 一键安装

  • 预装 Python 3.10+，通过脚本自动拉取代码、创建虚拟环境并启动服务。
  • 适合初学者快速上手。

🐳 Docker 部署

  • 适用于 Windows、Mac、Linux 系统，需执行文档中的三条指令。
  • 支持沙箱代码执行器，需额外配置 docker.sock。
  • 国内服务器需配置代理或使用国内镜像源。(AstrBot)

🔑 服务商配置

  • 支持 OpenAI、DeepSeek、Google Gemini 等服务商。
  • 可添加多个 API 密钥，并通过指令切换模型。

🧠 高级功能

🎭 人格设置

  • 可创建不同情景角色（如“小野猫、暖男”），动态调整机器人回复风格。

🔊 语音交互

  • 适配 TTS（文本转语音）和 STT（语音转文本），实现语音输入与输出。

📈 项目现状

  • 已获 9.3k+ Star，持续维护更新。
  • 提供完整文档和社群支持。
