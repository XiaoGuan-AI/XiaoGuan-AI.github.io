🎂 生日祝福网页 · Happy Birthday Webpage

一个温馨、可爱的生日网页，用代码传递心意，让祝福变得更特别 💖

✨ 项目简介

这是一个为特别的人制作的 个性化生日祝福网页，采用粉色系 UI 搭配插画、文字、动画与音效，呈现丰富又治愈的互动体验。

用户通过输入姓名和密码登录后，会看到：

  • 🎁 精心设计的祝福语
  • 🖼 插图与生日蛋糕图片
  • 🐾 趣味表情与漫画风格角色
  • 🎇 烟花动画效果
  • 📝 长文告白与深情寄语

🖥 页面结构

页面 功能说明
🎈 登录页 输入名字与密码后进入主页
💌 祝福页 展示一页页温馨祝福卡片
🎂 蛋糕页 可点击「喜欢 / 不喜欢」的小互动按钮
🎉 烟花页 动态烟花庆祝，呈现“生日快乐”字样
📖 最后一页 真挚的长文生日寄语（中英双语）

📷 页面展示

登录页 祝福卡片
烟花动画 长文祝福

💡 使用建议

  • 可部署到 GitHub Pages / Vercel / Netlify
  • 支持自定义文字、插图与动画
  • 适合作为送给朋友、恋人或家人的生日惊喜页面
  • 推荐在手机浏览器中访问，体验更佳

🛠 技术栈

  • HTML + CSS + JavaScript
  • 动态加载与卡片式 UI
  • 插画与动图增强页面情感表达

📬 自定义方法

你可以修改以下部分内容来自定义你的生日网页：

部分 文件/区域 内容
登录信息 index.html 中的输入框逻辑 修改验证用户名和密码
祝福语 data/cards.json 或 HTML 静态段落 替换为你想说的话
插图 assets/img/ 文件夹 可用自己绘制或收集的图片
动画 canvas.js / fireworks.js 替换烟花或动画脚本

🧁 特别鸣谢

感谢每一个努力让网页温暖起来的你。
愿所有美好的生日都不被辜负。🌈
