💖 浪漫爱心 · 相恋时间记录网页

🌐 项目地址：💖 浪漫爱心 · 相恋时间记录网页

✨ 功能亮点

功能 描述
❤️ 动态爱心 Canvas / CSS 绘制的心形脉动动画，支持渐变填充与呼吸效果
⏱️ 实时计时 JS setInterval 以毫秒级精度更新“已相爱时间”：xx 天 xx 时 xx 分 xx 秒 xx 毫秒
🖼️ 纪念照轮播 可添加多张合照，淡入淡出轮播展示
🎶 背景音乐 自动播放 & 循环播放；支持自定义歌曲、控制音量
📱 响应式布局 移动端 / PC 皆可完美居中显示
🛠️ 零依赖 纯原生 HTML + CSS + JavaScript，无框架、无外部库
🚀 一键部署 GitHub Pages / Vercel / Netlify 均可静态托管
