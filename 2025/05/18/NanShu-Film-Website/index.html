🎬 南属影视网

📍 作者： Jia’nan Zhao
📅 时间： 2025‑5‑23

🎯 bilibili项目演示地址： 🎬 南属影视网

欢迎访问 南属影视网，本网站是一个精致、分类明确的电影展示平台，支持多种浏览方式与信息查询，适合电影爱好者使用与学习参考。

🌟 网站简介

南属影视网（Nanshu Film）致力于为用户提供一个友好且高效的观影信息平台，用户可以：

  • 浏览电影封面与简介
  • 按分类或地区筛选电影
  • 查看评分排行与即将上映信息
  • 点击进入电影详情页，了解演员、导演与剧情介绍

🧭 网站导航

主导航栏涵盖以下电影类型：

分类子标签如：

  • 科幻片：外星人 / 自然灾难 / 生物变异
  • 爱情片：青春 / 文艺 / 喜剧等

📍 功能区域介绍

1. 首页头图轮播

  • 精选热映电影大图自动切换
  • 突出推荐影片（如：《超凡蜘蛛侠》《雷神》）

2. 排行榜与即将上映

  • 电影排行榜 Top5：展示评分最高的五部热播影片。
  • 即将上线影片：包含影片名与主角信息，提前预告即将更新的资源。

3. 电影信息卡片

  • 包含：电影封面、片名、主演、评分
  • 点击任意电影，进入详情页查看更多信息

🎞️ 电影详情页（示例）

点击影片《速度与激情：特别行动》进入详情页，可查看：

  • 🎬 导演：大卫·雷奇
  • 👥 主演：道恩·强森、杰森·斯坦森
  • 🎥 类型：动作片
  • 🌍 语言：英文
  • 🕒 发行时间：2019-8-2
  • 📖 剧情介绍：描述两位特工与“黑超人”对抗的精彩故事

🔍 分类筛选功能

用户可根据以下维度快速定位目标电影：

✅ 按地区：

大陆｜中国香港｜美国｜韩国｜法国｜日本｜英国｜德国｜印度｜其他

✅ 按类型：

剧情｜喜剧｜动作｜爱情｜惊悚｜犯罪｜悬疑｜科幻｜动画｜恐怖｜家庭｜战争｜传记｜冒险

✅ 按明星：

沈腾｜欧豪｜成龙｜李连杰｜甄子丹｜张晋 等

💡 技术亮点

  • 📄 使用 HTML + CSS + JavaScript 完成前端页面构建
  • 🧭 模块化设计，易于扩展与维护
  • 🖼️ 图片与资源管理整洁
  • 🖱️ 点击跳转逻辑流畅，用户体验良好

  • *.html：各类电影的独立展示页，可通过导航栏跳转访问。
  • check.php：可能用于登录验证或后台表单交互（如评论、注册等功能）。
