🔊 ESP32 蓝牙音箱制作教程（详细图文版）

📍 作者： Jia’nan Zhao
📅 时间： 2025‑6‑22

🎯 bilibili项目演示地址： 🔊 ESP32 蓝牙音箱

本教程将手把手教你如何使用 ESP32 + MAX98357A + 扬声器 自制一台支持 蓝牙连接手机外放音频 的音箱装置，适合电子爱好者、DIY 项目爱好者参考。

🧰 所需材料

名称 规格 / 型号 用途说明
ESP32 开发板 DevKit V1（30pin） 主控，支持蓝牙与 I2S 接口
MAX98357A 模块 I2S 数字音频解码放大器 将数字信号转换为模拟音频
扬声器 3W 4Ω～8Ω 音频输出单元
杜邦线 公对母 / 公对公若干 连接模块与开发板
Micro USB 线 用于给 ESP32 上电与烧录固件 电源与通信
热熔胶 / 胶带 — 固定模块用
外壳 / 容器 任意（如食品盒、木盒等） 保护并美化成品

📍 接线关系（ESP32 ←→ MAX98357A）

ESP32 引脚 MAX98357A 引脚 功能说明
5V VIN 电源正极
GND GND 电源地
GPIO22 DIN I2S 音频数据输入
GPIO26 BCLK I2S 位时钟
GPIO25 LRC (WS) I2S 左右声道时钟

🔊 扬声器连接

将 MAX98357A 模块上的扬声器输出口（SPK+ 和 SPK-）分别连接至扬声器的红黑线。

🔧 硬件组装步骤

实际安装参考图：

  1. 准备容器外壳：选择合适大小的塑料盒或金属盒子。
  2. 固定 ESP32：使用热熔胶将 ESP32 固定在盒盖背面，便于 USB 插拔。
  3. 固定 MAX98357A 与扬声器：模块与扬声器焊接后固定于盒底，并布好线。
  4. 穿孔走线：为音频线和 USB 线预留出口。
  5. 封闭装配：确保所有器件连接稳固、无短路后盖好盖子。

🔥 固件烧录步骤

✅ 使用工具：flash_download_tool_3.9.2

烧录界面截图如下：

📥 烧录所需文件与地址

文件名 烧录地址
bootloader.bin 0x1000
partitions.bin 0x8000
boot_app0.bin 0xE000
firmware.bin 0x10000

⚙️ 操作流程：

  1. 连接 ESP32 至电脑，选择正确的 COM 端口。
  2. 在 ESP32 下载工具中勾选四个 bin 文件并填写对应地址。
  3. 点击 START 开始烧录，看到 FINISH 完成 表示成功。
  4. 烧录完成后按下 ESP32 的 EN/Reset 键重启模块。

📲 蓝牙配对与使用

📡 启动蓝牙

默认使用 ESP32 A2DP Sink 蓝牙接收示例。

初始化代码示例：

#include "BluetoothA2DPSink.h"
BluetoothA2DPSink a2dp_sink;

void setup() {
  a2dp_sink.start("Bluetooth-Speaker-ESP32"); // 蓝牙设备名为 Bluetooth-Speaker-ESP32
}

void loop() {
  // 无需处理，自动接收音频数据
}

📱 手机连接蓝牙

  1. 打开手机蓝牙，搜索设备名 Bluetooth-Speaker-ESP32。
  2. 点击连接成功后，播放任意音乐。
  3. 音频会通过 ESP32 → MAX98357A → 扬声器播放出声音。

🎉 项目成果展示

  • ✅ 可连接手机蓝牙并播放音频。
  • ✅ 支持蓝牙音质 I2S 解码输出。
  • ✅ 成本低、制作简单、适合嵌入容器个性化定制。

📌 小贴士

  • 确保电源输出足够（推荐使用 ≥5V/1A）。
  • 盒内线缆尽量短，避免干扰。
  • 可加入 LED 指示灯、电池模块等扩展功能。
